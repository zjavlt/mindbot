# mindbot
Discord bot prototype for mental health therapy powered by Gemini 2.5 flash

## Features
- Gemini 2.5 flash model with custom instruction prompt and parsed patient notes
- Another model that generates patient notes in JSON format

### Availability
Gemini 2.5 flash is no longer available without subscription. 
Possible solution is to downgrade the Gemini model to the latest free version.
